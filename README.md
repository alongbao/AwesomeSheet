# AwesomeSheet

Frontend UIActionSheet

# Install

```shell
$ npm install awesomesheet
```

# Usage

```javascript

// require core javascript
var as = require('awesomesheet');

// require style
require('awesomesheet/lib/AwesomeSheet.scss'); // or css

as.show();
as.hide();
```

HTML:

```html
<div id="awesome-sheet">
  <div class="sheet">
    <div class="item">sheet 1</div>
    <div class="item">sheet 1</div>
    <div class="item">sheet 1</div>
  </div>
</div>
```
